# Bank Management System

A desktop-based banking application developed using Java Swing, JDBC, and MySQL.

## Features
- User Signup
- Login Authentication
- Deposit Money
- Withdraw Money
- Fast Cash
- PIN Change
- Balance Enquiry
- Mini Statement

## Technologies Used
- Java
- Swing & AWT
- JDBC
- MySQL

## Tools Used
- IntelliJ IDEA
- Git & GitHub

## Project Description
This project simulates basic banking operations with a graphical user interface and database connectivity using JDBC.
